How to embed the Kultrip Widget:

1. Include the widget CSS and JavaScript files:

<link rel="stylesheet" href="https://your-server.com/kultrip-widget.css"> <script src="https://your-server.com/kultrip-widget.umd.js"></script>

2. Create a container div with an ID:

<div id="kultrip-widget" style="width: 100%; height: 600px;"></div>

3. Initialize the widget:

<script> // Wait for the widget to load window.addEventListener('load', function() { if (window.KultripWidget) { window.KultripWidget.init('kultrip-widget'); } }); </script>
